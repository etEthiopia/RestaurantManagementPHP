------------------------------------------------------------
Restarant Reservation System V1.0
------------------------------------------------------------

This is a Restaurant Reservation System built using HTML, Bootstrap,
JQuery at the front end and PHP at the back end. 

-------------------------------------------------------

To start using the application: 


-------------------------------------------------------
	1. First import the database into your sql server
To do this, create a database named 'restaurant' and import the given sql database
file, also named 'restaurant.sql'. 
-------------------------------------------------------


-------------------------------------------------------
	2. Configure database connection
After completing step one, go to the 'models' folder and open 'db_connect.php'
and set the username and password of your sql server. 
-------------------------------------------------------


-------------------------------------------------------
 	3. After completing step two, please access the views folder in the browser. 
To do this, first connect your server, put the project folder
into the www directory (Wamp) and paste the following: 
	localhost/restaurant/views
into your browser search bar, or access the views folder using Index of /
Note that the 'index.php' file is located in the views folder. 
-------------------------------------------------------


After completing step three, the 'index.php' page should be displayed. If not, 
please return to step one. 





Using the application:


1. Navigate to the registration bar displayed in the header and click on register. 
2. After the registration page is displayed, enter your registration credentials, including your user type. 
3. If you select normal user, you should be able to proceed to registration. If you select 'administrator', you will
be prompted to enter an administrator code. Please enter 

	'admin'

as the administrator code. After this, you should successfully be able to register yourself as an administrator.
